# 🎥 Веб-приложение для видеозвонков

Простое и удобное веб-приложение для видеозвонков с возможностью отправки ссылок-приглашений.

## 🚀 Возможности

- **Видеозвонки в реальном времени** - WebRTC технология для качественной связи
- **Создание комнат** - генерация уникальных ID для приватных звонков  
- **Ссылки-приглашения** - отправка прямых ссылок для присоединения к звонку
- **Управление медиа** - включение/выключение камеры и микрофона
- **Демонстрация экрана** - возможность показать свой экран другим участникам
- **Адаптивный дизайн** - работает на всех устройствах

## 🛠 Технологии

- **Backend**: Node.js, Express, Socket.IO
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **WebRTC**: Для видео/аудио связи
- **Развертывание**: Railway

## 📦 Установка и запуск

### Локальная разработка

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер разработки:
```bash
npm run dev
```

3. Откройте браузер и перейдите на `http://localhost:3000`

### Развертывание на Railway

1. Подключите репозиторий к Railway
2. Railway автоматически определит настройки из `railway.json`
3. Приложение будет доступно по сгенерированному URL

## 🎯 Использование

1. **Создание звонка**: 
   - Нажмите "Создать комнату"
   - Скопируйте ссылку и отправьте участникам
   - Нажмите "Начать звонок"

2. **Присоединение к звонку**:
   - Введите ID комнаты или перейдите по ссылке
   - Разрешите доступ к камере и микрофону

3. **Управление во время звонка**:
   - 📹 Включить/выключить камеру
   - 🎤 Включить/выключить микрофон  
   - 🖥️ Демонстрация экрана
   - 🔗 Копировать ссылку на комнату
   - 📞 Завершить звонок

## 🔧 Конфигурация

Приложение использует публичные STUN серверы Google для NAT traversal. Для продакшена рекомендуется настроить собственные TURN серверы.

## 📱 Поддержка браузеров

- Chrome 60+
- Firefox 60+
- Safari 12+
- Edge 79+

## 🔒 Безопасность

- Все соединения используют HTTPS в продакшене
- WebRTC обеспечивает end-to-end шифрование медиа потоков
- Комнаты автоматически удаляются при отключении всех участников
