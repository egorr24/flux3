# 🐬 MySQL на Railway - ПРОСТАЯ НАСТРОЙКА

## Быстрая настройка (2 минуты)

### 1. Добавить MySQL
1. Открой свой проект на railway.app
2. Нажми **"Add Service"** 
3. Выбери **"Database"** → **"MySQL"**
4. Готово! Railway автоматически создаст все переменные

### 2. Проверить переменные
Railway автоматически создает эти переменные в твоем основном сервисе:
- `MYSQLHOST` - хост базы данных
- `MYSQLPORT` - порт (обычно 3306)  
- `MYSQLUSER` - пользователь
- `MYSQLPASSWORD` - пароль
- `MYSQLDATABASE` - название базы данных

### 3. Перезапустить
Просто перезапусти свой основной сервис - все заработает автоматически!

✅ Таблицы созданы/проверены

## Если не работает

1. **Проверь что MySQL сервис запущен** (зеленый статус)
2. **Подожди 1-2 минуты** после создания MySQL
3. **Перезапусти основной сервис** 
4. **Проверь логи** - там будет видно ошибку
