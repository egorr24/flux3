# 🚂 Быстрое развертывание на Railway

## Шаг 1: Добавить MySQL
1. Откройте ваш проект на [railway.app](https://railway.app)
2. Нажмите **"Add Service"** → **"Database"** → **"MySQL"**
3. Railway автоматически создаст все переменные

## Шаг 2: Добавить секретный ключ
1. Перейдите в настройки основного сервиса → **"Variables"**
2. Добавьте переменную:
   - **Имя**: `SESSION_SECRET`
   - **Значение**: `my-super-secret-key-12345` (замените на свой)

## Шаг 3: Проверить
После развертывания в логах должно появиться:
```
✅ MySQL подключен успешно!
✅ Таблицы созданы/проверены
✅ Сервер запущен на порту 3000
```

## 🎯 Готово!
Ваше приложение готово к использованию:
- Регистрация: `/register`
- Вход: `/login`
- Главная: `/`

## 🔧 Переменные Railway (создаются автоматически)
- `MYSQLHOST` - хост MySQL
- `MYSQLPORT` - порт (3306)
- `MYSQLUSER` - пользователь (root)
- `MYSQLPASSWORD` - пароль
- `MYSQLDATABASE` - база данных (railway)

Только добавьте `SESSION_SECRET` вручную!
