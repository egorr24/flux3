# 🚀 Быстрый старт

## Локальная разработка

1. **Установите MySQL** локально
2. **Создайте базу данных**:
   ```sql
   CREATE DATABASE videocall_app;
   ```

3. **Установите зависимости**:
   ```bash
   npm install
   ```

4. **Настройте .env** (уже создан):
   ```bash
   MYSQLHOST=localhost
   MYSQLPORT=3306
   MYSQLUSER=root
   MYSQLPASSWORD=your-password
   MYSQLDATABASE=videocall_app
   SESSION_SECRET=local-secret-key
   NODE_ENV=development
   ```

5. **Запустите сервер**:
   ```bash
   npm start
   ```

6. **Откройте браузер**: http://localhost:3000

## Railway развертывание

1. **Добавьте MySQL сервис** в Railway
2. **Добавьте переменную** `SESSION_SECRET` в настройках
3. **Готово!** Railway автоматически создаст все остальные переменные

## Использование

1. **Регистрация**: `/register` - создайте аккаунт
2. **Вход**: `/login` - войдите в систему  
3. **Главная**: `/` - создавайте комнаты и звоните
4. **Звонок**: `/call/room-id` - присоединяйтесь к звонку

## Функции

- ✅ Регистрация и авторизация через MySQL
- ✅ Создание комнат для видеозвонков
- ✅ WebRTC видео/аудио связь
- ✅ Управление камерой и микрофоном
- ✅ Демонстрация экрана
- ✅ Копирование ссылок-приглашений
- ✅ Сессии в базе данных
- ✅ Работает без авторизации (демо режим)

## Безопасность

- Пароли хешируются с bcrypt
- Сессии хранятся в MySQL
- HTTPS обязателен для WebRTC (Railway предоставляет автоматически)
- Валидация данных на клиенте и сервере
